The following guide will show you how to implement our API in other platforms, like Minestom or standalone.

Our API includes the `ControllerApi` and the `PlayerApi` class. The `ControllerApi` is used to interact with the SimpleCloud's [groups](/docs/manual/setup/groups), [servers](/docs/manual/setup/servers), and more, while the `PlayerApi` is used to interact with [players](/docs/manual/droplets/player).

## Basic Usage

Now that you have the dependencies, you can use the SimpleCloud API in your project.

```kotlin tab="Kotlin"
import app.simplecloud.controller.api.ControllerApi

suspend fun main() {
    val controllerApi = ControllerApi.createCoroutineApi()
    val groups = controllerApi.getGroups().getAllGroups()
    println(groups)
}
```

```java tab="Java"
import app.simplecloud.controller.api.ControllerApi;

public class Main {
    public static void main(String[] args) {
        ControllerApi controllerApi = ControllerApi.createFutureApi();
        controllerApi.getGroups().getAllGroups().thenAccept(groups -> {
            System.out.println(groups);
        });
    }
}
```
